# Portafolio de Desarrollo Web - FAiCOD

Este repositorio contiene la arquitectura base para mis proyectos de desarrollo web con Next.js 14. 

### Objetivos del Proyecto:
- Construcción de Landing Pages de alta velocidad.
- Integración de formularios de contacto con bases de datos.
- Pruebas de despliegue optimizado para clientes.

### Tecnologías:
- Next.js (App Router)
- Tailwind CSS
- Postgres
